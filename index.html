Stock Market Project

This is a Java project simulating a stock market, featuring classes like StockMarketProject.java and FlowerShares.java. It builds a simple stock trading environment where users can interact with shares and manage purchases through a GUI. The code handles stock transactions, displays information, and simulates market activities. The project demonstrates practical use of Java object-oriented programming concepts and graphical user interface design, making it a great showcase for applying Java skills in a real-world scenario.

It features critical scripts like manhwa_graber.py for data extraction and web_scap_chaplist.py for managing chapter lists. HPCG Testing

HPCG Testing and Artifact Generation is an ongoing research effort into using new and different tools in code development.
